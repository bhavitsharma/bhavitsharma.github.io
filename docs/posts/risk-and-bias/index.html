Note

We’re going to show that empirical risk is an unbiased estimator of the expected risk.

In statistics, if you have a quantity \(\theta\) that you want to estimate, but you can only estimate it via some fancy algorithms and random observations. Say you come up with the estimator \(\hat{\theta}\), then \(\hat{\theta}\) is an unbiased estimator of \(\theta\) if:

\[ \mathbb{E}[\hat{\theta}] = \theta \]

In general, bias of this estimate \(\hat{\theta}\) is defined as: \[ \text{Bias}(\hat{\theta}) = \mathbb{E}[\hat{\theta}] - \theta \]

Empirical Risk

We can’t compute the expected risk directly as we don’t know the true data distribution. Instead, we use the empirical risk, which is the average loss over the training data:

\[ R_{\mathcal{D}}(f) = \frac{1}{|\mathcal{D}|} \sum_{(x, y) \in \mathcal{D}} \mathcal{l}(f, (x, y)) \]

We’re assuming we have i.i.d samples from the data distribution.

Expected Risk

We want to find a predictive function \(f\) from some hypothesis space \(\mathcal{H}\) such that \(f(x) \approx y\) for all \((x, y) \in \mathcal{D}\), where \(\mathcal{D}\) is the training data, and it also minimizes the expected risk.

Lets say our loss function is \(\mathcal{l}(f, \mathcal{Z})\), where \(\mathcal{Z}\) is the data distribution.

The risk \(R(f)\) is defined as the expected loss over the data distribution \(\mathcal{D}\): \[ R(f) = \mathbb{E}_{\mathcal{Z}}[\mathcal{l}(f, \mathcal{Z})] \]

Expected Empirical Risk

Now we compute expected empirical risk by considering the distribution over all possible training sets \(\mathcal{D}\):

\[ \mathbb{E}_{\mathcal{D}}[R_{\mathcal{D}}(f)] = \mathbb{E}_{\mathcal{D}}\left[\frac{1}{|\mathcal{D}|} \sum_{(x, y) \in \mathcal{D}} \mathcal{l}(f, (x, y))\right] \]

Lets say \(\mathcal{D} = \{z_1, z_2, \ldots, z_n\}\), then the expected empirical risk is (because of linearity of expectation):

\[ \begin{align*} \mathbb{E}_{\mathcal{D}}[R_{\mathcal{D}}(f)] &= \frac{1}{n} \sum_{i=1}^n \mathbb{E}_{z_i}[\mathcal{l}(f, z_i)] \\ &= \frac{1}{n} \sum_{i=1}^n R(f) \\ &= R(f) \end{align*} \]

What does minimizing empirical risk mean?

Now, obviously during training, we don’t have the true data distribution \(\mathcal{Z}\), so we can’t minimize the expected risk directly. Instead, we minimize the empirical risk, which is an unbiased estimator of the expected risk. But what guarantees that minimizing the empirical risk will also minimize the expected risk?

For example, if you overfit the training data, then the empirical risk will be low, but the expected risk will be high.

Instead of abandoning the Empirical Risk Minimization (ERM) framework, we aim to refine it. We’ll explore conditions ensuring that ERM avoids overfitting. Specifically, we want conditions where good performance on training data by the ERM predictor likely indicates good performance on the overall data distribution.

A typical approach involves applying the ERM principle within a limited search space. Essentially, the learner pre-selects a set of possible predictors, known as a hypothesis class, denoted by \(\mathcal{H}\). Each hypothesis \(h \in \mathcal{H}\) maps inputs from \(X\) to outputs in \(Y\). Given a hypothesis class \(\mathcal{H}\) and a training sample \(S\), the \(\text{ERM}_{\mathcal{H}}\) learner applies the ERM principle to choose a hypothesis \(h \in \mathcal{H}\) that minimizes the error on \(S\). Formally, this is expressed as: \[ \text{ERM}_{\mathcal{H}}(S) \in \arg \min_{h \in \mathcal{H}} L_S(h), \] where \(\arg \min\) identifies the set of hypotheses in \(\mathcal{H}\) with the minimum error \(L_S(h)\) across \(\mathcal{H}\). By limiting the selection to hypotheses from \(\mathcal{H}\), we impose a bias towards certain predictors, referred to as inductive bias. This choice, made before seeing the training data, should ideally rely on prior knowledge about the problem at hand.

A key question in learning theory is identifying which hypothesis classes ensure \(\text{ERM}_{\mathcal{H}}\) does not lead to overfitting.

In the following sections, we’ll look at some conditions that ensure that minimizing the empirical risk will also minimize the expected risk.

Finite Hypothesis Class

We’re going to look at one proof which says we’re guaranteed to minimize ERM if we have a finite hypothesis class and our true generating function is in the hypothesis class. However, we’re going to make some strong assumptions for this proof, including the loss function and the problem setting.

Important

The assumptions are really important!! We’re going to relax them in the subsequent sections.

  1. We’re looking at a binary classification problem, where \(Y = \{-1, 1\}\).
  2. The loss function is defined as the fraction of misclassified examples. Let say our distribution is \(\mathcal{D}\) and \(S\) is a sampled training set from \(\mathcal{D}\), and we have a predictor \(h_S: X \to Y\) which is the output of the ERM algorithm. Then the loss function is defined as: \[ \mathcal{L_s}(h) = \frac{1}{n} \sum_{i=1}^n \mathbb{I}(h(x_i) \neq y_i) \] where \(\mathbb{I}\) is the indicator function.
  3. We’re assuming that the true generating function \(f\) is in the hypothesis class \(\mathcal{H}\).
  4. \(|\mathcal{H}|\) is finite i.e. the hypothesis class is finite.

Proof Sketch

We know we’ll pick the hypothesis \(h_S\) that minimizes the empirical risk for the training set \(S\). So \[ h_S \in \arg \min_{h \in \mathcal{H}} \mathcal{L}_S(h) \]

We can also pick a bad hypothesis \(h\) from the hypothesis class \(\mathcal{H}\) such that \(\mathcal{L}_S(h) = 0\) - for example - consider a hypothesis that predicts labels given in the training set as is. i.e.

\[ h_S = \begin{cases} y_i & \text{if there exists } i \in [n] \text{ such that } x_i = x \\ 0 & \text{otherwise} \end{cases} \]

This is saying if I get the same training example, I’ll predict the same label as in the training set, otherwise I’ll predict 0. This obviously is a bad hypothesis, but it has zero empirical risk.

We want to show in the proof that the probability of picking such a bad hypothesis is low (those who have 0 empirical risk) in our setting.

  1. Since, \(f \in \mathcal{H}\), we know that true risk of \(f\) is 0, which means when searching for the best hypothesis we can ignore any hypothesis that has a non-zero empirical risk.
  2. Now that we’ve picked the best hypothesis \(h_S\) who has an empirical risk of 0, there can be bad hypothesis \(h\) such that \(\mathcal{L}_S(h) = 0\) (an example is shown above) and good hypothesis \(h\) such that \(\mathcal{L}_S(h) = 0\).
  3. We’ll also denote a quantity \(\mathcal{L}_{\mathcal{D}, f}(h)\) which is the true risk of the hypothesis \(h\) with respect to the true distribution \(\mathcal{D}\) and the true generating function \(f\).
  4. Clearly for bad hypothesis \(h\), \(\mathcal{L}_{\mathcal{D}, f}(h) > 0\). Since our loss function is the fraction of misclassified examples, we’ll say that our hypothesis \(h\) is bad if \(\mathcal{L}_{\mathcal{D}, f}(h) > \epsilon\) for some \(\epsilon > 0\). \(\epsilon\) is the margin of error we’re willing to tolerate.
  5. This means that good hypothesis \(h\) will have \(\mathcal{L}_{\mathcal{D}, f}(h) \leq \epsilon\).
  6. Therefore, we’re interested in upper bounding the probability of picking a bad hypothesis \(h\).
  7. Points in the training sample \(S\) are drawn i.i.d from the true distribution \(\mathcal{D}\).

Proof

Given our proof outline, lets formalize the proof.

Important

Remember that we’re picking the best hypothesis \(h_S\) that minimizes the empirical risk for the training set \(S\). So it’s possible we might pick a bad hypothesis \(h\) such that \(\mathcal{L}_S(h) = 0\). It also means we might a good hypothesis \(h\) such that \(\mathcal{L}_S(h) = 0\).

We want to upper bound this quantity: \[ Pr[S : \mathcal{L}_{\mathcal{D}, f}(h_S) > \epsilon] \]

This basically says that the probability of the samples \(S\) such that the true risk of the hypothesis \(h_S\) is greater than \(\epsilon\) is upper bounded by some quantity.

Now lets say define the \(H_B\) as the set of bad hypothesis (all possible) \[ H_B = \{h \in \mathcal{H} : \mathcal{L}_{\mathcal{D}, f}(h) > \epsilon\} \]

Now lets consider the set of all possible training sets \(S\) which contains atleast one bad hypothesis \(h \in H_B\):

\[ M = \{S : \exists h \in H_B \text{ such that } \mathcal{L}_S(h) = 0\} \]

Now we’re going to use a trick in probability theory to upper bound the probability of \(M\). We can write \(M\) as a union over all bad hypothesis \(h \in H_B\):

\[ M = \bigcup_{h \in H_B} \{S : \mathcal{L}_S(h) = 0\} \]

Now we can use the union bound to upper bound the probability of \(M\):

\[ Pr[S: \mathcal{L}_{\mathcal{D}, f}(h_S) > \epsilon] \leq Pr[M] \leq \sum_{h \in H_B} Pr[\{S : \mathcal{L}_S(h) = 0\}] \]

This is because \(M\) is superset of all the events \(\{S : \mathcal{L}_S(h) = 0\}\). and also \(Pr(A \cup B) \leq Pr(A) + Pr(B)\).

Now, for a single hypothesis \(h \in H_B\), we can upper bound the probability of picking a training set \(S\) such that \(\mathcal{L}_S(h) = 0\) which means for training points \(x_i\) such that \(h(x_i) = y_i\) for all \(1 \leq i \leq n\).

Since \(h\) is a bad hypothesis, we know that \(\mathcal{L}_{\mathcal{D}, f}(h) > \epsilon\), this means the \(Pr[h[x_i] = y_i] \leq 1 - \epsilon\) for all \(1 \leq i \leq n\).

Now, combining all the above, we get: \[ Pr[S: \mathcal{L}_S(h) = 0] = \prod_{i=1}^n Pr[h(x_i) = y_i] \leq (1 - \epsilon)^n \]

So,

\[ Pr[S: \mathcal{L}_{\mathcal{D}, f}(h_S) > \epsilon] \leq \sum_{h \in H_B} (1 - \epsilon)^n \\ \leq |H_B| (1 - \epsilon)^n \]

Since \(|H_B|\) is finite, we can choose \(n\) large enough such that \((1 - \epsilon)^n\) is small enough and tends to 0.

Bias and Variance Tradeoff

Let \(x\) be fixed, \(y\) be “true” value, and \(\hat{y}\) be the prediction (assuming \(f^*\) is the best possible prediction and \(Y = f^*(x)\)). Then the expected risk can be decomposed in terms of bias and variance as follows.

If we consider that the training set \(\mathcal{D} = \{z_1, z_2, \ldots, z_n\}\) is a random quantity, then \(f^*\) is random and consequently, so is \(\hat{y}\). The expected risk can be decomposed as follows:

\[ \begin{align*} \mathbb{E}_{\mathcal{D}}(\hat{y} - y)^2 &= \mathbb{E}_{\mathcal{D}}(\hat{y}^2 -2\hat{y}y + y^2) \\ &= \mathbb{E}_{\mathcal{D}}(\hat{y}^2) - 2\mathbb{E}_{\mathcal{D}}(\hat{y}y) + y^2 \\ &= \mathbb{E}_{\mathcal{D}}(\hat{y}^2) - \mathbb{E}_{\mathcal{D}}(\hat{y})^2 + \mathbb{E}_{\mathcal{D}}(\hat{y})^2 - 2\mathbb{E}_{\mathcal{D}}(\hat{y}y) + y^2 \\ &= \text{Bias} + \text{Variance} \end{align*} \]
